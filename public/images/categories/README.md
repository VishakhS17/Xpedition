# Category Images

Place your category images in this folder with the following names:

## Required Images:

1. **adventure.jpg** - Image for Adventure category
2. **cruiser.jpg** - Image for Cruiser category
3. **roadster.jpg** - Image for Roadster category
4. **sport.jpg** - Image for Sport category
5. **touring.jpg** - Image for Touring category
6. **background.jpg** - Background image for the Browse by Category section

## Image Recommendations:

- **Category images**: Square images (400x400px or larger) work best for the circular category icons
- **Background image**: Wide image (1920x1080px or similar) for the section background
- **Format**: JPG, PNG, or WebP
- **Optimization**: Consider compressing images before adding them for better performance

## File Structure:

```
public/
  images/
    categories/
      adventure.jpg
      cruiser.jpg
      roadster.jpg
      sport.jpg
      touring.jpg
      background.jpg
```
